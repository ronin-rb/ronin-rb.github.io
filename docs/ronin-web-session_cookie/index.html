Description

ronin-web-session_cookie is a library for parsing and deserializing various session cookie formats. It supports Rack, Django (JSON and Pickled), and JWT.

Features

  • Supports the following session cookie formats:
    • Rack
    • Django (JSON and Pickle)
    • JSON Web Token (JWT)
  • Has 98% test coverage.
  • Has 97% documentation coverage.

  • python-pickle ~> 0.1
