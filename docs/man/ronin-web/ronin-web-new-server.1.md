## SYNOPSIS

`ronin-web new server` [*options*] [*FILE*]

## DESCRIPTION

Generates a new `ronin-web-server` script.

## ARGUMENTS

*FILE*
: The file to create.

## OPTIONS

`-H`, `--host` *HOST*\|*IP*
: Optional host or IP interface to listen on. Defaults to `0.0.0.0`.

`-p`, `--port` *PORT*
: Optional port to listen on. Defaults to `8000`.

`-h`, `--help`
: Print help information
