## NAME

ronin-payloads-list - Lists the available payloads

## SYNOPSIS

`ronin-payloads list` [*options*] *DIR*

## DESCRIPTION

Lists available payloads.

## ARGUMENTS

*DIR*
: The optional directory to list payloads from.

## OPTIONS

`-h`, `--help`
: Print help information
