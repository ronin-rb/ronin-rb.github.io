## SYNOPSIS

`ronin help` [*options*] [*COMMAND*]

## DESCRIPTION

Displays the list of available Commands or prints information on a specific
COMMAND.

## ARGUMENTS

*COMMAND*
: The optional command to get detailed help information on.

## OPTIONS

`-h`, `--help`
: Print help information
