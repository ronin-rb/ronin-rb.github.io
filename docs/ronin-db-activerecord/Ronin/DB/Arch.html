Overview

Represents a Computer Architecture and predefines many other common architectures (Arch.x86, Arch.x86_64, Arch.ppc, Arch.ppc64, Arch.mips, Arch.mips_le, Arch.mips_be, Arch.arm, Arch.arm_le, and Arch.arm_be).
