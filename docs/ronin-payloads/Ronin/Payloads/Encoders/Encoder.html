Overview

Base-class for all other payload encoders.

Example

# encoders/my_encoder.rb
module Ronin
  module Payloads
    module Encoders
      class MyEncoder < Encoder

        register 'my_encoder'

        def encode(data)
          return ...
        end

      end
    end
  end
end
